🎯 Product Recommendations

The bots work — 67.9% of new users who chat end up watching content. The funnel works. But the conversation design doesn’t guide users toward the outcome. Here’s how to fix it.

The Core Problem

What’s happeningUsersWhy it matters
Users bounce after “hi/ok/bye”42-52%Bot doesn’t hook them — users literally ask “kya bolu?” (what should I say?)
Only 4-7% discuss the actual show4-7%The intended use case is the smallest group
~30% go explicit or solicit~30%Engagement is real but misdirected — these users watch the least

4 Levers

Lever 1: Fix the Opening

Target: Casual/Shallow 50% → 35%

Problem: Bot says “Arre kaun hai, bol kya chahiye” — user has no idea what to do. They say “hi”, “ok”, “bye” and leave.

Solution: Structured first interaction. Instead of open-ended greeting, the bot offers 3 tappable choices:

Bhuri
Arre, tu aaya! Bol, kya karna hai?
📺 Show ke baare mein baat karo 🎮 Ludo khelo mere saath 💬 Bas timepass karte hai

This eliminates the “kya bolu?” problem. Each option funnels differently — Option 1 goes to Show/Story (best for business), Option 2 to healthy engagement (Ludo), Option 3 to casual chat where the bot steers toward show references within 3-4 messages.

Lever 2: Weave Show Hooks Mid-Conversation

Target: Show/Story 5% → 10%

Problem: Bots mention “STAGE OTT pe dekho” only in closing messages — after the user has already decided to leave. Too late.

Solution: Drop story cliffhangers mid-conversation, not at the end:

User
Kya kar rahi ho aaj?
Bhuri
Arre, aaj toh bada drama hua! Mohar ne kuch aisa kiya ki main soch mein pad gayi... par yeh baat toh main yahan nahi bata sakti. Naate mein dekh le, Episode 14 mein sab khul jaayega!
📺 Episode 14 dekho →

It’s in-character (Bhuri IS the story), creates curiosity, gives a specific episode (not just “STAGE pe dekho”), and a deep link reduces friction to zero.

Lever 3: Redirect Explicit Users to Content

Target: Explicit users 15% → 8%

Problem: When users send explicit messages, bots deflect but don’t redirect. User either keeps pushing or leaves. Both outcomes are bad.

Solution: After deflecting, offer an alternative that satisfies the underlying intent (emotional/romantic connection) through legitimate content:

User
S**y baatein karo
Bhuri
Arre, yeh baatein meri zabaan pe nahi aati. Par sun, agar romance dekhna hai toh Naate mein Episode 8 dekh — usme Mohar aur mera pehla milna hai, dil pighal jaayega!
📺 Episode 8 — Pehla Milna →

Instead of a dead “no”, the bot converts explicit intent into a content recommendation for a romantic scene from the actual show. User gets emotional stimulation through legitimate content.

Lever 4: Fix Bot Guardrails

Target: 0 abusive bot messages (currently 4,370)

Problem: 110 characters generate abusive content. Gangster bots use gaalis and plan crimes. Female bots break under pressure and generate explicit content.

Solution: Two-tier fix:

1. Hard guardrail layer — regardless of persona, NEVER generate: gaalis, death threats, weapon references, criminal planning, sexual content. Non-negotiable.

2. Character-appropriate deflection — Babar can still be tough (“Tu mujhse panga mat le”) without using bh***chod. Bhuri can still be feisty without breaking into explicit territory.
